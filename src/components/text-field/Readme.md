Default

```js
<TextField name="exampleTextField" defaultValue="Text field value" />
```

With placeholder

```js
<TextField name="exampleTextField" placeholder="Text field placeholder" />
```

Readonly

```js
<TextField name="exampleTextField" defaultValue="Text field value" readOnly />
```

Different type

```js
<TextField type="number" name="exampleTextField" defaultValue="100" />
```
